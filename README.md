Restaurant Management System (Full Stack)

This is a comprehensive web application designed to manage restaurant operations, including a user-friendly ordering interface and a robust administrative panel.🚀

Technologies Used

Frontend
React.js (Vite): For building a fast and interactive user interface.   
Bootstrap 5: Used for responsive styling and mobile-first design.   
JavaScript (ES6+): For handling dynamic functionality and API calls.   

Backend
PHP: Core server-side scripting. 

Database
MySQL: Relational database for storing user accounts, menu items, and orders.   

🛠️ Key Features
Admin Panel: Complete control over menu management and order tracking.   
User Panel: Interactive UI for customers to browse the menu and place orders.   
Real-time Ordering: Seamless ordering process with database synchronization.   
Secure Authentication: User and admin login system for data protection.
